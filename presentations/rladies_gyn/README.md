# Sobre

O curso de gglot2 foi um curso expositivo e o objetivo era responder duas perguntas sobre a redução da movimentação aérea durante a crise do Covid. A idéia é que essas perguntas sejam respondidas utilizando visualização de dados. Os scripts foram desenvolvidos usando o R e bibliotecas do tidyverse. As perguntas eram: 
1. Qual o percentual de queda da movimentação dos aeroportos na crise covid?
2. Houveram aeroportos com maior queda de movimentação do que outros?
